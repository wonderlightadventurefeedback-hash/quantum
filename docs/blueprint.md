# **App Name**: FinIntel AI

## Core Features:

- User Authentication & Profile Management: Secure user registration and login with JWT, offering a personalized dashboard to track learning progress, prediction history, portfolio data, and AI chatbot conversations.
- Financial Learning Paths: Access structured lessons covering core financial topics like stock market basics, risk management, and portfolio diversification, with user progress tracking.
- AI Stock Prediction Playground: An interactive tool for users to make stock movement predictions (up/down/confidence), compare them against a backend ML microservice's predictions, and receive an AI tool-generated explanation on why a prediction matched or failed. Displays historical accuracy and performance charts.
- AI-Powered News Intelligence: Fetches live financial news and performs AI-driven sentiment analysis (positive, negative, neutral). A visual dashboard displays sentiment gauges, correlation with market movements, and highlights bullish/bearish signals.
- Portfolio Analysis & Insights: Users input their investment holdings to generate comprehensive analyses, including asset allocation charts, risk exposure scores, sector diversification, and performance metrics, complemented by AI tool-suggested improvement insights.
- AI Financial Strategy Advisor: A contextual AI chatbot powered by the OpenAI API, providing personalized investment suggestions, risk warnings, market explanations, and finance Q&A. The tool leverages user-specific data like portfolio, prediction history, news sentiment, and learning progress.

## Style Guidelines:

- The visual scheme embraces a sophisticated, data-driven aesthetic with a dark background. The primary color, a rich purplish-blue (#6148DE), evokes intellect and modernity. The background is a deeply desaturated tone of the primary color (#1B1820) to maintain focus, while an energetic blue accent color (#68B9FF) highlights key interactions and data points.
- Headlines use 'Space Grotesk', a sans-serif font conveying a modern, tech-focused sensibility. Body text is set in 'Inter', a clean grotesque-style sans-serif for optimal readability. Code snippets or technical details will utilize 'Source Code Pro', a monospaced sans-serif.
- Clean, minimalist line icons are used throughout the application to complement the modern, fintech aesthetic, ensuring clarity and an uncluttered user interface, especially on glassmorphism elements.
- A responsive modern dashboard UI featuring structured sections. Key information and interactive elements are presented within 'Glassmorphism cards' to create a layered, visually dynamic, and contemporary feel.
- Subtle and smooth animations enhance user interaction, providing visual feedback for navigation, data loading, and interactive charts, aligning with a premium user experience.
